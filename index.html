Static Page with Multiple Polls

Poll with multiple votes options

Optional Attribute 'data-allow-retry' allows multiple votes. User may still vote after initial vote.

Poll with Single vote

By default attribute 'data-allow-retry' is false if not added. Allows to vote only once. Cannot be undo or reset.

Poll with same poll Id

Poll with same Id will not be rendered and throw error. This is to prevent same poll being used

Poll without proper configuration

Poll without proper mandatory 'data-poll' attribute will not be rendered. This also prevents new poll being added without mandatory data
